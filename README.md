# InVideo

## 工具介绍

1、基于 UE5 实现的 rtsp 的视频播放插件  
2、基于 UE5 实现的运行时输出 mp4 文件

### 使用方法

新建一个蓝图 widget 继承自 InVideoWgidget，并新建一个 Image 对象，重新命名为 ImageVideo。  
即可进行调用，可以参考 videoUmg 组件。
